MediAssist Setup Guide

Step 1: Create Supabase Account

  1. Go to https://supabase.com
  2. Click "Start your project"
  3. Sign up with GitHub or Email
  4. Create a new project (name it "rapidcare")

Step 2: Get Your API Keys

  1. In Supabase dashboard, go to Settings â†’ API
  2. Copy your Project URL (looks like: https://xxxxx.supabase.co)
  3. Copy your anon/public key (long string)

Step 3: Setup Database

  1. In Supabase, go to SQL Editor
  2. Click "New Query"
  3. Copy the SQL from database-schema.sql
  4. Paste and click "Run"

Step 4: Update Configuration

Edit /js/config.js and replace:

SUPABASE_URL: 'your-actual-url-here',
SUPABASE_ANON_KEY: 'your-actual-key-here',

Step 5: Test the Application

  1. Go to Home Page
  2. Click "Sign Up" to create an account
  3. Choose your role (Patient, Doctor, or Admin)
  4. Start using MediAssist!
